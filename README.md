# CheckAlarmNow (알리미)

데스크톱 알림을 놓치지 않도록 도와주는 Windows 데스크톱 펫 앱입니다.
화면 위에 상주하는 캐릭터가 미확인 알림이 있으면 점점 초조해하며 확인을 유도합니다.

## 주요 기능

- **알림 감시**: Windows 알림 센터의 알림을 실시간 모니터링
- **인내심 시스템**: 설정한 시간이 지나면 단계적으로 반응 (Idle -> Warn -> Alert)
- **포커스 감지**: 알림을 보낸 앱에 포커스하면 자동으로 확인 처리
- **아이콘 대포 이펙트**: Alert 상태 진입 시 아이콘 발사 이펙트
- **사운드 알림**: Alert 상태 진입 시 시스템 사운드 재생
- **모니터링 앱 필터**: 특정 앱의 알림만 감시하도록 설정 가능
- **트레이 아이콘**: 시스템 트레이에서 상태 확인 및 설정 접근
- **커스텀 이미지**: 상태별 펫 이미지를 사용자 지정 가능
- **자동 시작**: Windows 시작 시 자동 실행 옵션

## 설치 및 실행

### 빌드된 파일 사용

`dist/` 폴더의 실행 파일을 바로 실행합니다.

### 소스에서 빌드

```bash
# 빌드
dotnet build src/CheckAlarmNow/CheckAlarmNow.csproj -c Release

# 단일 파일로 퍼블리시
dotnet publish src/CheckAlarmNow/CheckAlarmNow.csproj -c Release
```

**요구사항**: .NET 7.0 SDK, Windows 10 (빌드 17763) 이상

## 설정

우클릭 메뉴 또는 트레이 아이콘에서 "설정"을 열 수 있습니다.

| 설정 | 설명 | 기본값 |
|------|------|--------|
| 인내심 | 알림 후 Alert까지 대기 시간 | 5분 |
| 모니터링 앱 | 감시할 앱 목록 (비어있으면 전체) | 전체 |
| 펫 크기 | 캐릭터 표시 크기 | 1.0x |
| 사운드 | Alert 시 사운드 재생 여부 | 켜짐 |
| 자동 시작 | Windows 시작 시 실행 | 꺼짐 |

### 인내심 프리셋

- **없음 (즉시)**: 알림 즉시 Alert
- **매우 낮음**: 1분
- **낮음**: 3분
- **보통**: 5분
- **높음**: 10분
- **매우 높음**: 15분
- **사용자 지정**: 직접 입력

## 상태 흐름

```
Idle (잠자기) -> [알림 감지] -> Warn (경고) -> [인내심 초과] -> Alert (알림!)
                                                                    |
Alert/Warn -> [해당 앱 포커스] -> Happy (확인 완료) -> [3초 후] -> Idle
```

## 사용법

- **드래그**: 좌클릭으로 캐릭터 위치 이동
- **설정**: 우클릭 > 설정
- **알림 리셋**: 우클릭 > 알림 리셋 (현재 감지된 알림 초기화)
- **종료**: 우클릭 > 종료
